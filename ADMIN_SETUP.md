# Admin Panel Setup Guide

## Создание админа

### 1. Запустите миграции базы данных

```bash
npm run db:migrate
```

Это выполнит все миграции, включая:
- Добавление колонки `is_admin` в таблицу `users`
- Добавление колонки `nickname` в таблицу `users`
- Создание админа с почтой `maksimgorkij21@gmail.com`
- Инициализацию платежных систем

### 2. Создайте админа (если миграция не создала автоматически)

```bash
npm run create-admin
```

Это создаст/обновит админа:
- **Email:** maksimgorkij21@gmail.com
- **Password:** admin123
- **Nickname:** admin

### 3. Войдите в систему

1. Откройте сайт
2. Нажмите "Join Now"
3. Введите email: `maksimgorkij21@gmail.com`
4. Введите пароль: `admin123`
5. После входа вы увидите кнопку "Admin" в Header
6. Нажмите на кнопку "Admin" для доступа к админ панели

## Функционал админ панели

### Управление пользователями

- Просмотр всех пользователей в таблице
- Изменение баланса пользователя (увеличить/уменьшить)
- Бан/разбан пользователя
- Подтверждение регистрации (если email не подтвержден)

### Управление платежными системами

- Просмотр всех платежных систем
- Включение/отключение платежных систем
- Редактирование данных платежных систем (API ключи, секреты)
- Удаление платежных систем

### Управление пулами ликвидности

- Просмотр всех активных пулов
- Фильтрация по сетям (BEP-20, ERC-20, SOL, UNISWAP, TRX)
- Добавление нового пула через Uniswap URL
- Удаление пулов

### Настройки сайта

- Настройка названия сайта
- Настройка комиссии
- Режим обслуживания

## Тестовый способ оплаты

Для тестирования добавлен тестовый способ оплаты:
- Выберите "Тестовый" при пополнении баланса
- Введите сумму
- Баланс будет пополнен мгновенно без реальной оплаты

## Примечания

- Все API endpoints защищены проверкой прав администратора
- Админ может быть создан только через миграцию или скрипт
- Пароль админа можно изменить через базу данных
